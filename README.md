# 🚀 Catalyst Crew

[![MERN Stack](https://img.shields.io/badge/Stack-MERN-blue.svg)](https://www.mongodb.com/mern-stack)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

**Catalyst Crew** is a high-performance community platform designed to empower innovators, facilitate collaboration, and accelerate project growth. Built with the powerful MERN stack, it offers a seamless and scalable experience for teams to thrive.

---

## 🛠️ Tech Stack

The application is built using the following core technologies:

| Technology | Purpose |
| :--- | :--- |
| **MongoDB** | NoSQL Database for flexible data storage |
| **Express.js** | Backend web framework for Node.js |
| **React.js** | Frontend library for building dynamic interfaces |
| **Node.js** | JavaScript runtime for server-side logic |

### 🎨 Design & Styling
- **CSS3 / Vanilla CSS**: For custom, premium UI components.
- **Modern Typography**: Utilizing clean, sans-serif fonts for readability.

---

## ✨ Features

- **User Authentication**: Secure JWT-based login and registration.
- **Dynamic Dashboards**: Real-time updates and interactive data visualization.
- **Collaboration Hub**: Tools for team communication and task management.
- **Responsive Design**: Fully optimized for mobile and desktop views.

## 🚀 Getting Started

### Prerequisites
- Node.js (v16+)
- MongoDB Atlas account or local installation
- Git

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/shrutinikam27/catalyst-crew.git
   cd catalyst-crew
   ```

2. **Backend Setup:**
   ```bash
   cd server
   npm install
   # Create a .env file with your MONGO_URI and PORT
   npm start
   ```

3. **Frontend Setup:**
   ```bash
   cd ../client
   npm install
   npm start
   ```
